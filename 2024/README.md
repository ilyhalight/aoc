## How to run

Replace <number> with number of day, e.g. day1

1. Install Bun.sh
2. Run your input in `day<number>/input.txt` (without ending with new line). If you want to check sample data, just remove input.txt
3. Run in terminal:

```
bun run day<number>/index.js
```
